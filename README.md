# Voxelized_Railway_Ballast_Generation

This repository contains the Python codes used to generate and assemble heterogeneous voxelized railway ballast models for ground-penetrating radar (GPR) simulations.

The repository includes:

- Python 3 codes for generating voxelized ballast sub-blocks
- Python 3 codes for assembling the final heterogeneous ballast domain
- Python 3 codes for visualization of the generated models
- Example input and output files for reproducibility

## Repository structure

### Code
Python scripts used for generating and assembling the ballast models.

### Input
3D stone geometry files (`.obj`) used to generate ballast particles.

### Output
Example outputs generated by the code, including voxel grids, visualizations, reports, and HDF5 datasets.

## Description

The workflow consists of two main scripts:

### Code_1.py
Generates voxelized ballast sub-blocks from procedurally placed 3D stone models.  
The script performs stochastic rigid-body placement and rotation of ballast particles and exports voxelized blocks in HDF5 format.

### Code_2.py
Loads the generated voxelized ballast blocks and assembles them into the final heterogeneous ballast domain used for numerical simulations.
