# Voxelized_Railway_Ballast_Generation

This repository contains the Python implementation used to generate and assemble heterogeneous voxelized railway ballast models for ground-penetrating radar (GPR) simulations.

The repository provides a reproducible workflow for constructing realistic ballast geometries and preparing them for numerical modeling.

## Repository structure

### Code

Python scripts used for generating and assembling voxelized ballast models.

### Input

3D stone geometry files (`.obj`) used to construct ballast particles.

### Output

Example outputs generated by the code, including voxel grids, visualizations, and HDF5 datasets.

---

## Description

The workflow consists of two main stages:

### Code_1.py — Generation of voxelized sub-blocks

This script generates voxelized ballast sub-blocks from procedurally placed 3D stone geometries.
It performs stochastic rigid-body placement and rotation of ballast particles and exports voxelized blocks in HDF5 format.

### Code_2.py — Assembly of heterogeneous ballast model

This script loads the generated voxelized sub-blocks and assembles them into a structured heterogeneous ballast domain suitable for numerical simulations.

## Installation

Clone the repository:

```bash
git clone https://github.com/Sepideh-Harajchi/Voxelized_Railway_Ballast_Generation.git
```

Navigate to the repository folder:

```bash
cd Voxelized_Railway_Ballast_Generation
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Usage

### Step 1 – Generate ballast voxel sub-blocks

```bash
python Code/Code_1.py
```

This step generates voxelized ballast sub-blocks and stores them in the `Output` directory.

---

### Step 2 – Assemble the final ballast model

```bash
python Code/Code_2.py
```

This step assembles the generated sub-blocks into a heterogeneous ballast model and produces visualization and output files.

## Notes
