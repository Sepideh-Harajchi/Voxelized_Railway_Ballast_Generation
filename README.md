# Voxelized Railway Ballast Generation

This repository contains the Python codes used to generate and assemble heterogeneous voxelized railway ballast models for ground penetrating radar (GPR) simulations.

Author: Sepideh Harajchi  
Department of Geoscience and Engineering  
Delft University of Technology, The Netherlands

## Repository structure

Code  
Python scripts used for generating and assembling the ballast models.

Input  
3D stone geometry files (.obj) used to generate the ballast particles.

Output  
Example outputs generated by the code, including voxel grids, visualizations, and HDF5 datasets.

## Description

The workflow consists of two main scripts:

Code_1.py  
Generates voxelized ballast sub-blocks from procedurally placed 3D stone models.  
The script performs stochastic rigid-body placement and rotation of ballast particles and exports voxelized blocks in HDF5 format.

Code_2.py  
Loads the generated voxelized ballast blocks and assembles them into the final heterogeneous ballast domain used for numerical simulations.

## Requirements

Python 3.x

Required Python libraries:

- numpy
- scipy
- matplotlib
- h5py

## Usage

Step 1 – Generate ballast voxel blocks

Run:

python Code/Code_1.py

This script generates 147 independent voxelized ballast sub-blocks and stores them in the Output directory.

Step 2 – Assemble the final ballast domain

Run:

python Code/Code_2.py

This script assembles the generated blocks into a structured heterogeneous ballast model and produces visualization and report files.
